## ◈  What Is This?

**AttritionIQ** is an end-to-end ML pipeline built on the IBM HR Analytics dataset. It predicts the probability that a given employee will leave the company, enabling HR teams to act proactively.


## ◉  Results at a Glance

<div align="center">

| Metric | Score | Context |
|:-------|:-----:|:--------|
| **Accuracy** | `82.31%` | On 20% held-out test set |
| **ROC-AUC** | `80.06%` | Key metric for imbalanced classification |
| **Precision (Leave)** | `45.10%` | Of predicted leavers, 45% actually left |
| **Recall (Leave)** | `48.94%` | Caught ~49% of all actual leavers |
| **F1-Score (Leave)** | `46.94%` | Harmonic mean of precision and recall |
| **CV Accuracy** | `87.33% ± 1.14%` | 5-fold stratified cross-validation |

</div>

> **Why ROC-AUC matters more than accuracy here:** The dataset is heavily imbalanced (84% "Stay", 16% "Leave"). A model that predicts "Stay" for everyone achieves 84% accuracy but catches zero at-risk employees. ROC-AUC of 80% means the model meaningfully discriminates between the two classes at any threshold.

## 🤖  Model Details

### Algorithm — Gradient Boosting Classifier

Builds decision trees sequentially — each tree learns from the residual errors of the previous. Ideal for tabular HR data: handles mixed types, robust to outliers, produces calibrated probabilities.

```python
GradientBoostingClassifier(
    n_estimators     = 300,      # depth of ensemble
    learning_rate    = 0.05,     # shrinks each tree's contribution
    max_depth        = 4,        # controls overfitting
    subsample        = 0.8,      # stochastic boosting — reduces variance
    min_samples_leaf = 20,       # minimum leaf size
    max_features     = "sqrt",   # feature subset per split
    random_state     = 42,
)
```

### Handling Class Imbalance

```python
from sklearn.utils.class_weight import compute_sample_weight

# Upweight the minority class (Attrition=Yes) during training
sample_weights = compute_sample_weight("balanced", y_train)
model.fit(X_train, y_train, sample_weight=sample_weights)
```

### Data Split

```python
# 80/20 stratified split — preserves 16% attrition ratio in both sets
train_test_split(X, y, test_size=0.20, stratify=y, random_state=42)
# → 1,176 train  ·  294 test
```

### Step 3 — Use programmatically

```python
import pickle, pandas as pd

# Load
with open("model_artifacts/model.pkl",          "rb") as f: model          = pickle.load(f)
with open("model_artifacts/label_encoders.pkl", "rb") as f: label_encoders = pickle.load(f)
with open("model_artifacts/feature_names.pkl",  "rb") as f: feature_names  = pickle.load(f)

CATEGORICAL_COLS = [
    "BusinessTravel","Department","EducationField",
    "Gender","JobRole","MaritalStatus","OverTime",
]

employee = {
    "Age": 29, "BusinessTravel": "Travel_Frequently", "DailyRate": 500,
    "Department": "Sales", "DistanceFromHome": 20, "Education": 2,
    "EducationField": "Marketing", "EnvironmentSatisfaction": 1,
    "Gender": "Male", "HourlyRate": 55, "JobInvolvement": 2,
    "JobLevel": 1, "JobRole": "Sales Representative", "JobSatisfaction": 1,
    "MaritalStatus": "Single", "MonthlyIncome": 2800, "MonthlyRate": 11000,
    "NumCompaniesWorked": 6, "OverTime": "Yes", "PercentSalaryHike": 11,
    "PerformanceRating": 3, "RelationshipSatisfaction": 2, "StockOptionLevel": 0,
    "TotalWorkingYears": 5, "TrainingTimesLastYear": 1, "WorkLifeBalance": 1,
    "YearsAtCompany": 2, "YearsInCurrentRole": 1, "YearsSinceLastPromotion": 0,
    "YearsWithCurrManager": 1,
}

df = pd.DataFrame([employee])
for col in CATEGORICAL_COLS:
    df[col] = label_encoders[col].transform(df[col])

probability = model.predict_proba(df[feature_names])[0][1]
print(f"Attrition risk: {probability*100:.1f}%")
# → Attrition risk: 74.3%
```
